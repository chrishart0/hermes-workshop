# WS-002 - Pre-work and prerequisites

## Status

Todo

## Priority

P0

## Goal

Create a simple attendee pre-work document that helps people arrive with Hermes installed, a model configured, and one idea ready.

## Scope

- Add a pre-work/prerequisites doc, likely `docs/pre-work.md`.
- Link to official Hermes install and setup docs instead of duplicating everything.
- Include a short checklist:
  - laptop and terminal
  - install Hermes
  - configure model/provider
  - run smoke test
  - optional gateway setup
  - bring one use-case idea
- Make clear that pre-work is optional but strongly recommended.

## Acceptance criteria

- [ ] Attendee can complete the basic Hermes setup before the workshop.
- [ ] The doc links to official Hermes docs for authoritative details.
- [ ] The doc includes a smoke test command.
- [ ] The doc includes model/provider guidance.
- [ ] The doc includes a short “bring one idea” section.
- [ ] README links to the pre-work doc.

## Notes

The private-company run went better because attendees had VMs with Hermes and keys preconfigured. This doc should recreate some of that advantage for a public workshop.
