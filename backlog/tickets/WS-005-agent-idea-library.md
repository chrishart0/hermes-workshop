# WS-005 - Agent idea library

## Status

Todo

## Priority

P1

## Goal

Create a browseable library of agent ideas attendees can use during the custom build lab.

## Scope

- Add a doc, likely `docs/agent-idea-library.md`.
- Include ideas for professionals, personal use, managers/founders, developers, sysadmins, researchers, writers, and local/community use.
- Make each idea concrete enough to start building.

## Suggested format per idea

```markdown
## Agent idea

- **Who it helps:**
- **What it watches:**
- **What it knows about you:**
- **What it produces:**
- **First safe version:**
- **Stretch version:**
```

## Acceptance criteria

- [ ] Includes at least 10 strong agent ideas.
- [ ] Includes non-programmer/professional examples.
- [ ] Includes personal-life examples.
- [ ] Includes technical/sysadmin examples.
- [ ] Each idea has a first safe version.
- [ ] README and/or static site links to the idea library.

## Notes

This supports the “art of the possible” section and gives attendees somewhere to go when they ask, “Okay, but what should I build?”
