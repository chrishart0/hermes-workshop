# WS-006 - Real-world agent examples

## Status

Todo

## Priority

P1

## Goal

Research and curate real-world examples of people using agents or agent-like workflows effectively.

## Scope

- Search blogs, GitHub, X/Twitter, Reddit, Hacker News, and similar sources.
- Prefer concrete examples over hype.
- Include professional, personal, and technical use cases.
- Summarize the useful pattern attendees can steal.

## Suggested format per example

```markdown
## Example title

- **Who it is for:**
- **What the agent does:**
- **Why it is useful:**
- **Workshop takeaway:**
- **Source:**
```

## Acceptance criteria

- [ ] Includes curated source links, not just invented examples.
- [ ] Includes examples beyond programming/coding agents.
- [ ] Each example has a workshop takeaway.
- [ ] The doc is useful during the opening “art of the possible” phase.
- [ ] Good examples are candidates for slide additions.

## Notes

This should become a library of strong samples, not a generic link dump.
