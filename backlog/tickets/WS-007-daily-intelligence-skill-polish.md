# WS-007 - Daily intelligence skill polish

## Status

Todo

## Priority

P1

## Goal

Improve the Daily Intelligence Report skill bootstrap so attendees answer it more easily.

## Scope

- Revise confusing language, especially “What is your world?”
- Add examples for different attendee types.
- Keep the four-question bootstrap simple.
- Preserve the self-improving skill pattern.
- Keep the skill read-only and evidence-first.

## Acceptance criteria

- [ ] Bootstrap questions are easier to understand.
- [ ] “Your world” is replaced or clarified with concrete language.
- [ ] Examples include professional, personal, and technical contexts.
- [ ] The skill still asks only four questions.
- [ ] The first-run prompt remains clear.

## Notes

People liked the bootstrapping skill, but “your world” threw some attendees. The fix is language and examples, not a more complicated flow.
