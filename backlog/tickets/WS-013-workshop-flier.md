# WS-013 - Workshop flier

## Status

In progress

## Priority

P1

## Goal

Create a simple printable flier to advertise the Hermes workshop around town.

## Scope

- Decide whether a flier is worth doing for this event.
- If yes, create a clean one-page design suitable for coffee shops, coworking spaces, campuses, maker spaces, and community boards.
- Include the public site URL / QR code, title, date/time/location, who it is for, what attendees will build, and bring-a-friend language.
- Use Hermes-inspired styling without making the flier too visually noisy.

## Acceptance criteria

- [ ] Decision made: make flier or skip.
- [ ] If making it, produce printable PDF and source file.
- [ ] Include QR code to `hermes.arcadian.cloud` or final event URL.
- [ ] Include concise value proposition and attendee requirements.
- [ ] Verify print readability in black-and-white or low-quality print conditions.

## Notes

This is secondary to the public site and outreach copy, but potentially useful for local promotion.
