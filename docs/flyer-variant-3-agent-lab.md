## Canva layout notes

- Use the generated asset as the full-bleed background.
- Keep the Hermes landing-page aesthetic: electric blue `#0000FF`, pure white type/art, no gradients, no robot imagery.
- Set the headline in a high-contrast serif such as Playfair Display, Bodoni 72, Didot, or Libre Baskerville.
- Set details in Inter, Manrope, Montserrat, Aptos, or another clean sans.
- Place headline in the upper-left/center-left negative space.
- Place event details beneath the subhead in a compact white text block.
- Put the QR code in a white card near the lower-left or lower-center with a quiet margin.
- Place AI Collective and Hermes logos in a calm bottom row; use a white/black footer card only if needed for logo contrast.
