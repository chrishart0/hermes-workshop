# Four-Hour Workshop Outline

<p class="ol-lead">The full shape of the day: what happens, when you build, and what you walk out with. Optional pre-work before you arrive, then five focused phases. Every one ends with something running.</p>

<dl class="ol-meta">
  <div><dt>Total time</dt><dd>About 4 hours, plus optional pre-work.</dd></div>
  <div><dt>Format</dt><dd>Hands-on - you build, we help.</dd></div>
  <div><dt>You leave with</dt><dd>A real agent running on your machine.</dd></div>
</dl>

## The shape of the day

<ol class="ol-phases">
  <li class="ol-phase">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 0</span>
      <span class="ol-phase-time">Before · async</span>
    </div>
    <div class="ol-phase-body">
      <h3>Pre-work</h3>
      <p>Get Hermes installed and a model configured before you arrive if you can. Not mandatory, but every minute here becomes build time later. <a href="/pre-work">Open the checklist →</a></p>
      <p class="ol-deliver"><span>You leave with</span> Hermes running on your machine.</p>
    </div>
  </li>
  <li class="ol-phase">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 1</span>
      <span class="ol-phase-time">45-60 min</span>
    </div>
    <div class="ol-phase-body">
      <h3>The art of the possible</h3>
      <p>See what practical agents can do today, then choose a useful idea worth building. This comes first on purpose: examples make setup feel worth it.</p>
      <p class="ol-deliver"><span>You leave with</span> A concrete idea to build.</p>
    </div>
  </li>
  <li class="ol-phase">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 2</span>
      <span class="ol-phase-time">30-45 min</span>
    </div>
    <div class="ol-phase-body">
      <h3>Hermes setup</h3>
      <p>Install Hermes, configure a model, run a smoke test, and set up an optional chat delivery path.</p>
      <p class="ol-deliver"><span>You leave with</span> A working CLI and a live chat gateway.</p>
    </div>
  </li>
  <li class="ol-phase ol-phase--mark">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 3</span>
      <span class="ol-phase-time">45-60 min</span>
      <span class="ol-phase-tag">Shared build</span>
    </div>
    <div class="ol-phase-body">
      <h3>Daily Intelligence Agent</h3>
      <p>Everyone builds the same baseline agent together, end to end. It gives the whole room the same successful first loop before anyone branches into custom work.</p>
      <p class="ol-deliver"><span>You leave with</span> A shared agent that reads, reasons, and reports.</p>
    </div>
  </li>
  <li class="ol-phase ol-phase--mark">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 4</span>
      <span class="ol-phase-time">75-90 min</span>
      <span class="ol-phase-tag">Your build</span>
    </div>
    <div class="ol-phase-body">
      <h3>Custom build lab</h3>
      <p>The heart of the longer format, and the longest block for a reason. Point the pattern at your own project, data, or workflow and build your own agent, with help on hand throughout.</p>
      <p class="ol-deliver"><span>You leave with</span> Your own rough-but-real agent.</p>
    </div>
  </li>
  <li class="ol-phase">
    <div class="ol-phase-meta">
      <span class="ol-phase-n">Phase 5</span>
      <span class="ol-phase-time">30 min</span>
    </div>
    <div class="ol-phase-body">
      <h3>Demos, voting &amp; prize</h3>
      <p>Show what you built, the room votes on favorites, and a prize goes out. Rough-but-real agents are exactly what we want to see.</p>
      <p class="ol-deliver"><span>You leave with</span> A demo, a vote, and a winner.</p>
    </div>
  </li>
</ol>

## You build twice

<div class="ol-builds">
  <div class="ol-build">
    <p class="ol-build-label">Phase 3 · Shared</p>
    <p>Everyone builds the <strong>Daily Intelligence Agent</strong> together: one guided path so the whole room lands the same first win.</p>
  </div>
  <div class="ol-build">
    <p class="ol-build-label">Phase 4 · Yours</p>
    <p>Then you build <strong>your own agent</strong> in the lab, adapting the pattern to a real job in your world.</p>
  </div>
</div>

## Make the most of it

<ul class="ol-tips">
  <li>Arrive with the <a href="/pre-work">pre-work</a> done. Setup time turns into build time.</li>
  <li>Bring a real, repetitive task from your week. The build lab is best spent on something you actually want.</li>
  <li>Pair up. Two people split the setup friction and usually land a better idea.</li>
</ul>

<div class="ol-after">
  <p class="ol-after-label">After the workshop</p>
  <p>You keep everything you built. Take the pattern back to your own work, and use the <a href="https://hermes-agent.nousresearch.com/docs">official Hermes docs</a> as you go further.</p>
</div>
